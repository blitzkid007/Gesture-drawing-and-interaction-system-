# Gesture-drawing-and-interaction-system-
🎨 AI Air Canvas (Pure Version)

一个极简主义的 AI 空中画板，基于 MediaPipe 手势识别技术。

无需键盘鼠标，只需对着摄像头捏合手指，即可在空中挥毫泼墨。

✨ 特性 (Features)

🖐️ 纯粹手势交互：基于 MediaPipe Hands，毫秒级响应。

🚀 零依赖纯本地：无后端、无 AI 生成 API、无数据上传，保护隐私。

📷 智能画面适配：自动裁切适配屏幕比例，拒绝画面拉伸，实现沉浸式“照镜子”体验。

🖌️ 丝滑笔触：内置贝塞尔曲线平滑算法与手势防抖逻辑。

⚡ 对象消除：智能橡皮擦，触碰笔画即可整条消除。

🎮 如何使用 (How to Use)

启动：打开网页，允许摄像头权限。

绘画：

伸出食指：移动光标。

捏合食指与拇指：开始书写/绘画。

松开手指：停止书写。

工具：

🖌️ 画笔：默认工具，支持多种颜色。

⚡ 橡皮擦：移动到笔画上即可消除。

🗑️ 清空：一键清空画布。

🛠️ 技术栈 (Tech Stack)

HTML5 / CSS3 / JavaScript (ES6+)

Google MediaPipe Hands

HTML5 Canvas API
